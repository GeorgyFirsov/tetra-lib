◆ tea2_step()

uint8_t tea2_step ( TEA2_CONTEXT *  context)

Performs one gamma generator step.

Note, that TEA2 skips 50 steps in the beginning and 18 steps between each pair of subsequent gamma bytes used for encryption! I.e. the following algorithm implements actual TEA2 encryption:

int skip_rounds = 18;
for (int i = 0; i < 50 - skip_rounds; ++i)
tea2_step(&context);
for (int i = 0; i < plaintext_length; ++i)
{
for (int j = 0; j < skip_rounds; ++j)
tea2_step(&context);
ciphertext[i] = plaintext[i] ^ tea2_step(&context);
}
